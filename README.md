# Proyecto de Predicción de Ventas de Autos

Este proyecto tiene como objetivo optimizar el stock y la rentabilidad de un concesionario premium mediante el uso de modelos predictivos.

## Descripción
- **Modelo de Rentabilidad**: Estima el precio de venta de vehículos para identificar oportunidades de mayor margen.
- **Forecasting de Demanda**: Proyecta las ventas mensuales para mejorar la planificación del stock.

## Requisitos
- Python 3.8+
- Librerías necesarias (ver archivo `requirements.txt`)
